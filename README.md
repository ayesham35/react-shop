# React Shop

A product catalog built with pure React.

## Features

- Product grid with search and pagination
- Product detail view
- Shopping cart with quantity controls
- Cart persists after page refresh (localStorage)
- JWT authentication via DummyJSON
- User session persists after page refresh

## Tech Stack

- React 18
- Vite
- Plain CSS
- Browser Fetch API
- Browser localStorage
- DummyJSON (free fake REST API)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Test Login

Use any dummy user from [dummyjson.com/users](https://dummyjson.com/users)

Example: `emilys` / `emilyspass`
